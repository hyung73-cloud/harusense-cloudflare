# Harusense Service Expansion Guide

이 문서는 하루센스 기관 상세페이지에 수액, 보톡스, 예방접종 같은 품목을 안정적으로 늘릴 때의 기준입니다.

## 현재 공개 원칙

- GLP-1 가격 지도는 기존처럼 전체 공개합니다.
- 수액, 보톡스, 예방접종 상세 항목은 현재 뉴센스의원만 시범 공개합니다.
- 다른 기관에는 아직 추가 진료 항목을 노출하지 않습니다.
- 메인 지도, 로그인, QR, 가격 저장 로직과 분리해서 관리합니다.

## 품목 추가 흐름

1. `data/treatments.catalog.json`에 품목을 먼저 등록합니다.
2. 가격 입력 키는 `price_key_prefix`를 기준으로 만듭니다.
3. 실제 가격 데이터가 있는 기관에서만 상세페이지에 노출합니다.
4. 충분히 검증되기 전에는 `seo_enabled`와 `map_enabled`를 켜지 않습니다.
5. 뉴센스의원에서 먼저 보이고 저장되는지 확인한 뒤 전체 공개 여부를 결정합니다.

## 공개 전 검사

업로드 묶음을 만들면 아래 검사가 자동 실행됩니다.

- 품목 카탈로그와 입력 화면의 키가 일치하는지 확인
- 뉴센스의원에만 추가 진료 항목이 보이는지 확인
- 다른 기관에 추가 항목이 노출되지 않는지 확인
- 사이트맵 주소가 깨지지 않았는지 확인
- 메인 지도, 로그인, QR, API, Apps Script 파일이 포함되지 않았는지 확인

## 전체 공개 전 체크

전체 기관에 추가 진료 항목을 열기 전에는 반드시 아래를 확인합니다.

- 관계자 입력 화면에서 저장이 빠르게 되는가
- 상세페이지에서 가격이 있는 품목만 보이는가
- 가격이 없는 품목이 빈 카드로 노출되지 않는가
- 모바일에서 카드가 너무 길어지지 않는가
- 기존 GLP-1 가격 지도 표시가 흔들리지 않는가

## 원칙

품목은 한 번에 많이 열지 않습니다. 뉴센스의원에서 먼저 확인하고, 문제가 없을 때 하나씩 전체 공개합니다.
