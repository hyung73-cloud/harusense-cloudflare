# 하루센스 SEO 5차: 기관 허브 페이지 보강 결과

생성일: 2026-06-10

## 목적

`/clinic/` 페이지를 단순 기관 목록에서 기관 상세페이지와 지역 가격페이지를 함께 묶는 SEO 허브로 보강했습니다.

## 수정 내용

- `/clinic/` 제목과 설명 강화
- 요약 카드 추가: 기관 상세페이지 14개, 지역 가격 페이지 9개
- 지역별 가격 바로가기 추가
- 기관별 상세페이지 목록을 더 읽기 쉬운 카드형 구조로 정리
- JSON-LD CollectionPage 유지/보강

## 건드리지 않은 것

- 메인 지도 앱
- 지도핀/가격/재고 표시 로직
- 로그인/권한신청/비밀번호 변경
- QR 방문 확인
- Apps Script URL
- Cloudflare D1/API 코드
- sitemap.xml
